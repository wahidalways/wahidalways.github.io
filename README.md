# Md. Wahiduzzaman Nayem - Portfolio Website

A professional portfolio website for Business Analyst Md. Wahiduzzaman Nayem, built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth animations
- **Interactive Elements**: Hover effects, smooth scrolling, and dynamic content
- **Contact Form**: Functional contact form that opens email client
- **Performance Optimized**: Fast loading with optimized images and code

## Sections

1. **Hero Section**: Introduction with profile image and call-to-action buttons
2. **About**: Personal background and key statistics
3. **Experience**: Professional timeline with detailed work history
4. **Skills**: Technical and soft skills with progress bars
5. **Projects**: Featured projects with descriptions and technologies used
6. **Contact**: Contact information and functional contact form

## Technologies Used

- HTML5
- CSS3 (with Flexbox and Grid)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts (Inter)
